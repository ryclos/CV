  • Automatisation des renouvellements de certificats
    AZURE VIRTUAL MACHINE WINDOWS SERVER POWERSHELL
    ▼
    • Script

    Un certificat Let's Encrypt gratuit, qui doit être mis à jour tous les deux mois car il expire après trois mois, était renouvelé manuellement. Lors de mon arrivée, j'ai rédigé un script en PowerShell permettant l'automatisation de ce processus. Ce script applique également les certificats dans le Keystore Java des applications QMatic, qui fonctionnent sous Java. De plus, il met à jour les certificats pour une application front QMatic, en les reliant symboliquement à ceux du répertoire Certbot et en modifiant leur format de PEM12 en CERT et KEY.

  • Déploiement d'un script batch pour mode kiosk
    WINDOWS 11 POWERSHELL
    ▼
    • Script

    Comprendre le besoin d'une entreprise souhaitant déployer des bornes interactives sur leurs différents sites de vente. Chaque borne se composait d'un mini PC et d'un écran, avec un système d'exploitation Windows 11 sur lequel était installée l'application Intuiface. Pour que l'application fonctionne en mode borne (kiosk) et soit accessible au public tout en empêchant toute tentative d'accès au système sous-jacent, un script batch a été développé. Ce script crée un utilisateur Windows standard avec un mot de passe, configure le mode kiosk et modifie le registre (regedit) pour un autologon avec cet utilisateur. L'application Intuiface est alors lancée en mode shell à la place de explorer.exe, assurant ainsi une expérience utilisateur sécurisée et ininterrompue.
